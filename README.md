# Slopworks

Post-apocalyptic base-building survival game. Reclaim real buildings by restoring their mechanical systems, build Satisfactory-style automation networks, and defend your territory from hostile fauna.

**Engine:** Unity
**Players:** 1-4 (solo + co-op)
**Platform:** Desktop

## What Makes It Different

Every reclaimable building is modeled from real BIM data -- real duct layouts, real pipe routing, real mechanical rooms.
